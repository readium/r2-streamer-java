landmarks nav

[REQUIRED] epubnav-040 Access

Tests whether the landmarks nav can be accessed.

If the Reading System provides a means to access the landmarks (access to the first page of content and the glossary), the test passes.

If this test fails, all subsequent tests for the landmarks-nav should be marked Not Supported.

[REQUIRED] epubnav-050 Document-level linking

Tests whether linking to Content Documents from the landmarks nav is possible.

To complete this test, bring up the landmarks and access the link labeled "Index". (If the Reading System provides a quick link to the index, that is also acceptable.)

If a document with the title "Index" loads, the test passes.

[REQUIRED] epubnav-060 Fragment-level linking

Tests whether linking to fragments within Content Documents from the landmarks nav is possible.

To complete this test, bring up the landmarks and access the link labeled "Glossary". (If the Reading System provides a quick link to the glossary, that is also acceptable.)

If the heading "Glossary" appears on the page that loads, the test passes.
